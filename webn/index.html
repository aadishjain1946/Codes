Karvonen Calculator

The Karvonen Formula is used to find the target heart rate of a person. It is calculated by multiplying table percentage value and heat rate reserve plus resting heart rate.

karvonen Value = [Heart Rate Reserve * Table Percentage Value + Resting Heart Rate]


Generally, we should use Maximum Heart Rate and resting heart rate to calculate Karvonen value but if the Maximum Heart Rate value is not given and instead of its value of age is given then we can also calculate Karvonen value.

karvonen Value = [220 - age - Resting Heart Rate] ---(for male)

karvonen Value = [226 - age - Resting Heart Rate] ---(for female)
